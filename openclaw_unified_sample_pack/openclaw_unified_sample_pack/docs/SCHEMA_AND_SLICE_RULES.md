# 字段与切片说明

## 统一样本原则
1. 所有开发、演示、试运行都基于本样本包。
2. 正式切换生产数据时，只替换 `templates/` 对应结构的数据，不改变字段和目录。
3. `01_dev_core_december` 用于开发与演示。
4. `02_holdout_validation_november` 用于 Week 3 新样本/留出验证。
5. `03_stress_regression_aug_sep` 用于压力回归。
6. `04_baseline_transition_apr_jul` 用于背景理解与规则解释。
7. `05_formal_trial_live_placeholder` 用于生产数据替换。

## 重要边界
- 2025-06-23/06-27 之前的周报，没有同周期的“进展中项目台账快照”；这些样本应主要用于文本理解与规则解释。
- 量化指标按“截至某日”快照存放，已对齐到对应周报结束日期。
- 结项/中止表为静态目录，不作为逐周快照，但可用于规则校验和历史归档。

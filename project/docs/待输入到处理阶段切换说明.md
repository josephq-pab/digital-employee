# 待输入到处理阶段切换说明

**版本：** v1.0
**日期：** 2026-04-08
**阶段：** 阶段4P《等待期维护与催办节奏管理》

---

## 一、切换逻辑总览

| 收到输入 | 切换到 |
|---------|-------|
| 评审反馈 | 评审反馈受理记录.md → FB-001 登记 |
| 真实样本 | 真实样本受理记录.md → S-001 登记 |
| 两者都到 | 并行处理 → 评审线+样本线并行 |

---

## 二、收到评审反馈后的切换

```
触发：收到评审反馈
       ↓
更新《评审反馈受理记录》FB-001
       ↓
更新《评审反馈分类与收口表》（判断类型）
       ↓
更新《扩大评审状态更新说明》
       ↓
标记分支（A/B/C）
       ↓
如需修订 → 输出《扩大评审修订动作清单》
如已形成结论 → 输出《扩大评审反馈处理结果》
```

**处理入口文档：**
- 评审反馈受理记录.md
- 评审反馈分类与收口表.md
- 扩大评审状态更新说明.md

---

## 三、收到真实样本后的切换

```
触发：收到真实样本
       ↓
更新《真实样本受理记录》S-001
       ↓
执行《真实样本校验记录》6项校验
       ↓
更新《真实样本差异说明》
       ↓
判断结论：
  通过 → 输出《真实样本校验结论》+《影子运行进入建议》
  不合格 → 输出《真实样本补充要求清单》
```

**处理入口文档：**
- 真实样本受理记录.md
- 真实样本校验记录.md
- 真实样本差异说明.md

---

## 四、两者都到后的切换

```
触发：评审反馈 + 真实样本均到达
       ↓
并行处理：评审线 + 样本线（各自走上述流程）
       ↓
并行完成 → 输出《双线输入处理汇总》
       ↓
判断汇合条件 → 评审线与样本线汇合条件.md
       ↓
如满足 → 输出《影子运行启动前状态确认》
       ↓
触发条件全部满足 → 进入影子运行启动阶段
```

---

## 五、切换动作锁定

**以下动作已锁定，收到输入后直接执行，无需重建流程：**

| 输入类型 | 切换动作 |
|---------|---------|
| 评审反馈 | FB-001登记 → 分类 → 状态更新 → 分支标记 |
| 真实样本 | S-001登记 → 校验 → 差异记录 → 结论输出 |
| 两者都到 | 并行处理 → 汇总 → 汇合判断 → 影子运行判断 |

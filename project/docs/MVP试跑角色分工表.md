## 一、角色定义

| 角色 | 代称 | 职责描述 |
|------|------|---------|
| 样本提供方 | 样本负责人 | 负责提供/更新周报、台账、映射文件 |

## 三、职责边界说明

### 3.1 各角色禁止事项

| 角色 | 禁止事项 |
|------|---------|
| 运行负责人 | 不得擅自修改规则逻辑；不得跳过冒烟测试；不得将provisional标记改为正式版 |
| 复核负责人 | 不得在未核实原始数据的情况下判定输出"正确"；不得遗漏问题登记 |
| 问题登记人 | 不得将P0降级为P1/P2；不得删除问题记录；不得隐瞒阻断性问题 |
| 阶段判断人 | 不得在有P0的情况下给出A/B级结论 |
| 联络负责人 | 不得将外部确认结果自行假定为"已确认"，必须等正式书面确认 |

### 3.2 协作流程

```
样本提供 → 运行负责人执行 → 输出归档 → 复核负责人检查
       ↓
  问题登记人登记问题
       ↓
  阶段判断人给出结论
       ↓
  联络负责人推进外部确认
```

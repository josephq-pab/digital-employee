## 五、发现阻断性问题时的处理规则

**如果收到E类（阻断）问题：**

1. 立即记录到 `project/docs/PROBLEM_LOG.md`
2. 评估是否触发冻结建设版禁止项
3. 不得在无证据的情况下声称"E类问题已排除"
4. B1状态更新为：⏳ 有E类问题（具体记录）

**如果收到B/C类反馈（非阻断但有偏差）：**

1. 记录具体偏差内容
2. 评估是否属于P1/P2
3. 不得在未处理的情况下关闭相关项

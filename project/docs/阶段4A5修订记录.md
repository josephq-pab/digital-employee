## 二、关键状态机定义

### 阻断项状态机（8种状态）

```
未发出 → 已发出 → 已签收 → 待反馈
                 ↘         ↙
                  反馈中 → 已通过 → 已关闭
                           ↕
                      被退回
```

### 门控卡颜色定义

| 颜色 | 含义 | 条件 |
|------|------|------|
| 🟢 | 可进入4B | 全部G为🟢 |
| 🟡 | 可继续准备但不能开工 | G1-G5任一为🔴但已发出 |
| 🔴 | 绝对不能进入4B | G1-G5任一仍为🔴未发出或未到位 |

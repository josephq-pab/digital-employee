## 四、推荐确认路径（最小路径）

**目标：最少确认次数进入4B**

```
第一步：发 MGT-01 给管理层（1页，5分钟可读完）
        ↓
第二步：发 PMO-01+PMO-02+ADPT-02+ADPT-03 给 PMO（合并为一个包）
        ↓
第三步：发 PILOT-01 给 PMO+管理层
        ↓
前三步完成后 → 门控 G1/G2/G3 → 🟢
        ↓
第四步（并行）：发 ADPT-01 给 PMO（如有意见）→ 不阻断4B
        ↓
进入4B实现
```

**最小必须确认：MGT-01 + PMO-01 + PMO-02 + ADPT-02 + ADPT-03 + PILOT-01（共6项）**
**暂缓项可在试点前或试点中确认，不阻断4B开工**

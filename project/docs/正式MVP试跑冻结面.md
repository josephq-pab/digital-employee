## 六、试跑期微调申请流程

如需微调（不涉及冻结项）：
```
申请微调 → Agent评估 → 判断是否涉及冻结项
    → 不涉及冻结项 → 执行微调 → 记录到周报
    → 涉及冻结项 → 暂停试跑 → 申请Q01/Q03/Q04重新确认 → 确认后更新冻结面
```

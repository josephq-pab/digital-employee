**维护人：** PMO协调员（PC）
**更新频率：** 收到新样本时更新

---

## 一、样本分级说明

| 级别 | 数量 | 用途 | 到位标准 |
|------|------|------|---------|
| S1 | ≥1份 | 阶段1关闭 | 1份S1通过验收，即为"到位" |
| S2 | ≥3份 | 阶段4进入 | 3份S2全部通过验收，方为"到位" |
| S3 | 试点期连续 | 阶段5 | 试点期间持续提交，不设数量上限 |

## 六、PC样本工作日常检查项

**每次样本流转时，PC必须：**

- [ ] 收到新样本后立即登记
- [ ] 脱敏状态确认（FU已完成脱敏）
- [ ] 执行5维验收
- [ ] 登记验收结果
- [ ] 如通过：更新registry.json，通知Agent
- [ ] 如退回：立即通知FU补充
- [ ] 更新本跟踪表

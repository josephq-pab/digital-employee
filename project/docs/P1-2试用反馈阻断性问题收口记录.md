## 二、反馈类型定义（供参考）

| 类型 | 代码 | 说明 |
|------|------|------|
| 体验问题 | A | 页面操作不顺畅/看不懂/找不到 |
| 表达问题 | B | 文字表述不清楚/有歧义/不像正式材料 |
| 流程问题 | C | 操作流程不对/与实际业务不符 |
| 非阻断问题 | D | 问题存在但不影响核心判断 |
| **阻断性问题** | **E** | 导致无法正常使用或判断完全错误 |

**E类（阻断）问题包括但不限于：**
- 结论卡数字与原始周报明显不符（P0级数据错误）
- 台账空白无法使用
- 页面完全不可用
- 判断逻辑严重错误（如把延期写成达标）

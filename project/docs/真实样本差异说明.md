## 一、差异类型定义

| 类型 | 定义 | 对影子运行的影响 |
|------|------|----------------|
| 正常业务差异 | 业务表述不同但结构一致 | 不影响，可适配 |
| 适配层问题 | 字段命名或格式不同，需调整适配器 | 可修复，≤1人天则修正 |
| 阻断性问题 | 存在结构性缺陷，无法适配 | 退回业务方，重新提供 |

---

## 二、预期可能存在的差异

### 2.1 正常业务差异（预期存在，不阻断）

| 差异项 | 说明 | 适配方式 |
|--------|------|---------|
| 字段命名 | 如 alert_lines 可能命名为 risk_items | 适配器已支持字段映射 |
| 数字格式 | 如期间标识格式可能为"第XX期" | 不影响解析 |
| 中文数字 | 如"两"可能写作"2" | 适配器已支持中文数字 |

### 2.2 适配层问题（需评估修复成本）

| 差异项 | 说明 | 修复方式 |
|--------|------|---------|
| 字段完全缺失 | 关键字段不存在 | 评估是否可从 raw_text 提取 |
| 嵌套结构 | 数据结构与预期不同 | 评估修复复杂度 |

### 2.3 阻断性问题（会导致退回）

| 差异项 | 说明 | 处置 |
|--------|------|------|
| 字段完全不可用 | request_lines / alert_lines / raw_text 均无 | 退回，要求提供有内容的周报 |
| JSON无法解析 | 文件损坏或格式错误 | 退回，要求重新提供 |
| 内容为测试数据 | 无实际业务内容 | 退回，要求提供真实周报 |

---

## 三、差异评估流程

```
样本到位 → 执行校验 → 记录差异项 → 分类差异类型
                                            ↓
                              阻断性 → 退回业务方
                              适配层问题 → 评估修复成本
                                           ≤1人天 → 修正适配器
                                           >1人天 → 评审确认是否继续
                              正常业务差异 → 记录，继续
```

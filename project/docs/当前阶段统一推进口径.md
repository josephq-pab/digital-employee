
## 5A阶段口径补充

**当前阶段**：并行运行期（2026-04-09起）

### 第1句：什么是并行运行

```
Q：数字员工现在在做什么？
A：与现有业务机制并行运行。
   接收同一份PMO周报，生成领导摘要、事项台账、风险清单，
   由人工对照现有机制复核，不替代人工决策。
```

### 第2句：不争论大方向，只对具体问题决策

```
Q：现在还在讨论"这个方向对不对"吗？
A：不再讨论。
   从现在开始，只对具体问题做ABC决策：
   有具体案例 → 进入决策表 → 推荐A/B/C → 决策人拍板
   无具体案例 → 不进入决策流程
```

### 第3句：按价值推进

```
Q：怎么判断要不要继续？
A：按价值判断。
   每周期记录：输入/输出一致率、偏差点、是否减轻工作、是否产生管理价值。
   4周后做综合判断：价值明显→扩大范围 / 部分价值→保留改进 / 无价值→暂停
```

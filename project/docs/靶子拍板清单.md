## 问题3：如果只改一轮，优先改哪一类

**含义**：假设只能再修改一次页面，您希望优先改什么。

**三个选项（只能选一个）**：
- **样式**：颜色、字体、布局、排版等视觉呈现
- **结论表达**：领导摘要里的文字判断、措辞、口径
- **台账与风险呈现**：表格结构、风险ID命名、历史追踪展示方式

**判断维度**：
- 样式是最表层的问题，改动最简单但影响也最小
- 结论表达是核心问题，措辞是否像业务人员写的
- 台账与风险呈现是实用性验证，决定了业务方会不会真的用

```
您的选择：☐ 样式   ☐ 结论表达   ☐ 台账与风险呈现
```

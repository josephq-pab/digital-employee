## 六、首检通过标准

**全部24项必须全部通过（24/24），方可视为迁移成功。**

若有任何一项未通过：
1. 记录未通过项编号和实际情况
2. 停止服务：`pkill -f http.server`
3. 检查文件是否完整传输
4. 重新启动服务并复检
5. 若问题无法解决，联系原服务器负责人

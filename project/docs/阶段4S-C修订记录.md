## 四、关键新增机制

### 4.1 演示前/中/后完整闭环

```
演示前7项（P1-P7）→ 演示中7项（D1-D7）→ 演示后6项（A1-A6）
全部打勾 = 演示才算完成
```

### 4.2 收口三问

```
问题1：是否认可当前输出结构 → 同意/有条件同意/暂缓
问题2：是否同意真实样本到位后进入影子运行 → 同意/有条件同意/暂缓
问题3：是否同意继续推进扩大评审准备 → 同意/有条件同意/暂缓
```

### 4.3 演示后按结论分类动作

```
结论=同意 → 直接进入下一阶段
结论=有条件同意 → 列出条件 → 处理 → 确认
结论=暂缓 → 记录阻断 → 分析 → 短时可解/长时阻塞
```

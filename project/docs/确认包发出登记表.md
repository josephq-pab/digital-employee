**原则：按事实记录，不得伪造"已发出"**

## 一、登记说明

| 状态值 | 含义 | 说明 |
|--------|------|------|
| 待发出 | 尚未发出 | 包在Agent/用户手中，未实际发送 |
| 已发出 | 已发送给接收方 | 以发出日期为准 |
| 已签收 | 接收方已收到 | 以签收日期为准 |
| 待反馈 | 签收后等待反馈 | 以期望反馈日期为参考 |
| 已反馈 | 收到确认反馈 | 以反馈日期为准 |
| 已确认 | 全部同意/修改后同意 | 进入门控更新流程 |

## 六、状态更新规则

1. **每次发送后1个工作日内** 更新发送日期和状态（待发出→已发出）
2. **每次签收确认后1个工作日内** 更新签收日期和状态（已发出→已签收）
3. **每次收到反馈后1个工作日内** 更新反馈结果和状态（已签收→已反馈/已确认）
4. **状态变化后立即同步** 更新《阶段4B进入门控卡》

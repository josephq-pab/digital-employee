## 七、真实数据来源确认的前置问题

**在确认数据来源前，必须先回答以下问题：**

1. **周报数据从哪里来？**
   - 现有周报系统导出的JSON？
   - 人工录入？
   - 其他系统API？

2. **台账数据从哪里来？**
   - 现有台账系统导出的JSON？
   - 人工维护的表格？
   - 其他系统API？

3. **映射关系怎么处理？**
   - 方案A：系统按项目名称 fuzzy 匹配（类似现有sample_loader的find_ledger_record逻辑）

   - 方案C：业务系统提供标准化ID对接

4. **数据的更新频率是什么？**
   - 每周更新一次？
   - 实时更新？
   - 按需更新？

## 使用说明

每执行一次影子运行，填写一行记录。登记表为持续累积文档，每次真实样本接入后追加记录。

---

## 字段说明

| 字段 | 说明 |
|------|------|
| 登记序号 | 自动编号（RUN-001开始）|
| 登记日期 | 执行影子运行的日期 |
| 样本期数 | 样本覆盖的期数 |
| 样本时间范围 | 样本的时间跨度 |
| 输入事项数 | 人工判断的总事项数 |
| 适配成功数 | 适配器成功处理的项数 |
| 适配成功率 | 适配成功数 / 输入总事项数 |
| 系统识别正确数 | 系统判断 = 人工判断的项数 |
| 系统识别错误数 | 系统判断 ≠ 人工判断的项数 |
| 遗漏数 | 人工判断有，系统未识别的项数 |
| 误报数 | 系统识别，人工判断不存在的项数 |
| 差异率 | (识别错误数 + 遗漏数) / 输入总事项数 |
| 差异项详情 | 记录差异项的具体描述 |
| 差异原因分类 | 适配器问题 / 规则问题 / 样本问题 |
| 结论 | 可接受 / 需修正 / 暂停 |
| 处理动作 | 针对差异项的具体处理 |
| 登记人 | 执行人 |

## 结论判定标准

| 差异率 | 结论 | 建议动作 |
|--------|------|---------|
| ≤ 10% | 可接受 | 继续扩大验证范围 |
| 10%~20% | 需修正 | 修正适配器后继续验证 |
| > 20% | 暂停 | 问题修复后重新评审 |

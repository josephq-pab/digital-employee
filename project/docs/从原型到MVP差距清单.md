## 一、总体结论

**当前演示版到MVP，还差4类东西：**
1. 还未解决的问题（QUAL-03/04）
2. 必须确认的外部口径（Q01/Q03/Q04）
3. 必须补强的实现能力
4. 可以延后的能力

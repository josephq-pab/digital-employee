## 二、字段说明

| 字段 | 说明 |
|------|------|
| 序号 | 自动编号（S-001开始）|
| 样本名称 | 样本的文件名或批次名 |
| 周期 | 样本覆盖的周报期间 |
| 提供方 | 哪个业务团队提供 |
| 提供方联系人 | 具体对接人 |
| 目标到位日期 | 业务方承诺的提供日期 |
| 实际到位日期 | 实际提供的日期（到位后填写）|
| 是否脱敏 | 是/否/部分（到位后填写）|
| 是否符合模板 | 是/否（到位后填写）|
| 状态 | 待提供 / 已到位 / 已校验 / 已拒绝 |

---

## 三、状态说明

| 状态 | 说明 |
|------|------|
| 待提供 | 业务方尚未提供 |
| 已到位 | 样本已接收，待校验 |
| 已校验 | 校验通过，可进入影子运行 |
| 不合格 | 校验不通过，需重新提供 |
| 已拒绝 | 样本提供方主动拒绝提供 |

---

## 四、催收触发规则

| 触发条件 | 动作 |
|---------|------|
| 目标到位日期到期未到位 | 发送提醒至提供方联系人 |
| 提醒后3个工作日未到位 | 升级通知至PMO负责人 |
| 5个工作日未到位 | 记录延迟原因，评估是否继续等待 |

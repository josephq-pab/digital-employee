## 二、本阶段口径工作

### 2.1 W17战略客户标注范围明文化（G1缺口）

**问题：** W17战略客户数据标注v2已落地，但"战略客户标注范围"的判断依据未明文化到DATA-001。

**当前状态：**
- ✅ W17期战略客户标注已完成（v2强化标注）
- ⚠️ 但"哪些客户属于战略客户"的判断依据未写入DATA-001

**本阶段处理：**

在 DATA-001 登记册中补充：

```
战略客户标注范围判断依据（明文化）
- 依据来源：指标文件中的人民币贷款余额前20大客户
- 判断标准：余额最高的前20名客户定义为战略客户
- 标注方式：在样本中用「战略客户」标签显式标注
- W17特殊说明：W17期指标文件中战略客户指标数据未更新（518.1亿与W16相同）
  → 标注时使用"实际发生"逻辑，不依赖指标文件的时点值
  → 在数据接入页提示"战略客户数据待业务确认"
```

**关闭标准：** 判断依据写入 DATA-001，页面增加"战略客户数据待业务确认"提示。

### 2.2 M-002 W17数据待更新问题（G1相关）

**问题：** W17期战略客户口径贷款余额518.1亿元，与W16期完全相同，疑似样本数据缺失。

**当前状态：**
- ⚠️ W17期 M-002 数据与 W16 完全相同（518.1亿元）
- 分析结论：属于样本数据缺失，不是系统错误
- 阶段6J已在数据接入页显式标注

**本阶段处理：**

补充以下内容到 DATA-001：

```
W17期M-002数据异常说明
- 现象：W17期战略客户口径人民币贷款余额518.1亿，与W16完全相同
- 原因分析：W17原始样本包中战略客户指标数据未更新
- 系统处理：在数据接入页显示"数据待更新"，结论卡标注"数据待确认"
- 不是系统错误，不影响系统冻结判断
- 冻结后如果接真实数据，需确认该指标数据是否正确接入
```

**关闭标准：** W17数据问题有明确记录，结论卡有"数据待确认"标注。

### 2.3 样本数据与生产数据边界（G2缺口）

**问题：** 用户可能把样本数据当作真实数据。

**当前状态：**
- ⚠️ MVP页样本切换后，全局无"样本数据"明显提示

**本阶段处理：**

在 DATA-001 中补充：

```
样本数据与生产数据边界说明
- 样本模式：当前所有可切换的8个样本均为样本数据，不是真实生产数据
- 样本用途：支撑演示、验证逻辑、推进冻结版建设
- 生产模式：接入真实业务数据后的运行状态
- 区分方式：在样本切换区增加"【样本数据】"角标
- 冻结后接入真实数据时，需从MVP页移除样本角标，切换到生产模式
```

**关闭标准：** MVP页样本切换区域有"【样本数据】"角标，DATA-001有明文记录。

## 五、禁止"持续观察"要求

**本阶段不允许写"持续观察"作为结论。**

每个口径问题必须有：
- ✅ 明确原因（可定位）
- ✅ 明确影响（可量化）
- ✅ 明确处理方式（本阶段关闭 or 迁移后处理）

如果无法给出明确结论，必须升级为 P1 问题，触发冻结建设版禁止项检查。

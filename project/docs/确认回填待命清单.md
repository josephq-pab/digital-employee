## 三、G1升级的触发条件（重要）

**G1升级条件：PMO-01 + PMO-02 + ADPT-02 + ADPT-03 全部"同意"或"修改后同意"**

这4项是AND关系，不是OR关系：
- 任一"不同意" → G1不升级
- 4项全部同意 → G1→🟢
- 部分暂缓 → G1维持🟡，其余项确认后可升级

## 五、收到反馈后的标准动作（5步）

```
Step 1：识别反馈来源（哪个确认包）
Step 2：识别结果类型（同意/修改后同意/不同意/暂缓）
Step 3：执行本清单中对应结果行的"立即动作"
Step 4：更新《确认结果回填模板》对应编号
Step 5：更新《阶段4B进入门控卡》对应G项
```

**收到任一反馈，5分钟内完成Step 1-3，不积压。**

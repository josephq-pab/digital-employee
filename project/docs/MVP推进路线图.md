## 六、MVP验收标准

MVP验收通过需满足：

| 标准 | 说明 |
|------|------|
| 准确率≥80% | PMO对判断结果认可 |
| 及时性 | 每期输出在收到周报后24小时内完成 |
| 可追溯 | 每项判断都有规则依据 |
| 管理层认可 | 领导摘要格式被管理层接受 |
| 持续运行 | 连续3期试点无重大阻断 |

## 七、阶段5A：并行运行与价值验证（2026-04-09新增）

### 5A阶段目标

建立并行运行机制，进入按周运行、按期记录、按问题决策、按价值推进的状态。

### 5A阶段核心机制

| 机制 | 说明 |
|------|------|
| 并行运行 | 与现有业务机制并行，不替代人工 |
| 按周运行 | 周一接数据 → 周五产出+复核 → 周五下午收口 |
| 价值验证 | 每周期记录输入/输出一致率、偏差点、减负效果、管理价值 |
| ABC决策 | 只接受具体问题，不接受抽象评价 |
| 切换判断 | 连续4周满足4个条件后，可升级为局部替代 |

### 5A关键文档

| 文档 | 用途 |
|------|------|
| `并行运行机制说明.md` | 当前运行方式 |
| `并行运行周期安排.md` | 每周节奏 |
| `价值验证记录表.md` | 每周期填写 |
| `具体问题ABC决策表.md` | 具体问题决策记录 |
| `从并行运行到局部替代的判断条件.md` | 升级/降级标准 |

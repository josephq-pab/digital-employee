## 三、登记表更新规则

1. **发送后**：立即登记发送日期、发送人、接收人
2. **签收后**：立即登记签收日期
3. **收到反馈**：当日内登记反馈日期、反馈内容摘要
4. **确认通过**：当日内更新状态，并更新《决策记录与待确认事项.md》
5. **退回修改**：立即通知发送责任人

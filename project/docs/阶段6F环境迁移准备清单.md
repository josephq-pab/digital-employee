## 三、新环境依赖

| 依赖项 | 说明 |
|--------|------|
| Python 3.x | 用于启动http.server |
| 无需Node.js | 纯前端单文件，无构建步骤 |
| 无需数据库 | 所有数据内嵌在HTML中 |
| 无外部CDN依赖 | CSS/JS均内联 |

---

## 四、启动方式

```bash
# 进入页面目录
cd <页面目录>

# 启动HTTP服务（任意端口）
python3 -m http.server 3000

# 访问
# http://<服务器IP>:3000/
```

---

| 项 | 当前值 | 迁移后需确认 |

## 六、验证迁移完整性的检查项

迁移到新环境后，必须验证：
- [ ] 页面可正常打开（http://新IP:端口/）

- [ ] 领导摘要有内容

## 三、收到反馈后的填写规范

一旦收到评审反馈，按以下规范填写：

| 字段 | 填写规范 |
|------|---------|
| 状态 | 收到后改为"已收到" |
| 反馈日期 | 实际评审会议日期 |
| 反馈人 | 评审参与人姓名 |
| 反馈类型 | 认可型 / 条件型 / 修订型 / 暂缓型 |
| 反馈内容 | 评审意见摘要（原文记录，不加工）|
| 是否影响扩大结论 | 是 / 否 |
| 建议处理动作 | 根据反馈类型确定（见《评审反馈分类与收口表》）|
| 登记人 | 技术侧登记人姓名 |

## 一、优化原则

1. ✅ 不新增页面
2. ✅ 不改主结构（HTML结构不变）
3. ✅ 只允许：文案优化 / 显示逻辑优化 / 关键交互顺序优化 / 关键数据展示优化
4. ✅ 所有优化必须真正落到页面
5. ❌ 不允许只写计划不改页面

## 四、每包发出后的标准动作

```
确认包发出
    │
    ▼
登记到《确认包签收与反馈登记表》
    │
    ▼
等待接收人签字反馈
    │
    ├── 超过期望反馈时限 → 发起第1次催办
    │    ├── 催办话术：见《阻断项催办话术与节奏.md》
    │    └── 催办记录：登记到《确认包签收与反馈登记表》
    │
    └── 收到反馈
         ├── 通过 → 更新《决策记录与待确认事项.md》状态
         ├── 修改后通过 → 更新对应文档 → 通知相关方
         ├── 需重提 → 退回 → 重新发出
         └── 暂缓 → 记录 → 继续催办
```

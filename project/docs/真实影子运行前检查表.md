## 五、全部通过后

| 动作 | 说明 |
|------|------|
| 执行影子运行 | 按《真实生产数据影子运行SOP》第1~5步执行 |

---

## 六、任一检查项不通过的处置

| 不通过项 | 处置 |
|---------|------|
| A1~A3（样本格式问题）| 退回业务方，说明不合格项 |
| B1~B3（样本质量问题）| 退回业务方，要求补充或重新提供 |
| C1~C2（适配器问题）| 评估修正工作量，≤1人天则修正，否则暂停 |
| C3（对接人问题）| 暂停，等待业务方确认 |
| C4（登记表问题）| 技术侧立即处理 |

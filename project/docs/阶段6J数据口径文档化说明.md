## 三、M-001 口径说明（已冻结）

| 字段 | 内容 |
|------|------|
| 指标名称 | 汽车零部件贷款余额净增 |
| 单位 | 亿元 |
| 含义 | 报告期内汽车零部件行业新增授信已提款余额 |
| 计算方式 | 当前值 - 基期值（基期为空则从0开始） |
| 判断规则 | gap = current - target；gap<0为缺口，gap≥0为达标 |
| 状态阈值 | 缺口≥5亿→红；缺口<5亿→橙；缺口<3亿→黄；达标→绿 |
| 数据来源 | 指标文件（如指标文件缺失则显示"数据待更新"） |
| **冻结状态** | **✅ 已冻结（口径明确，无争议）** |

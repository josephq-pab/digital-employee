## 二、演示目标

1. **建立认知**：让听众理解数字员工在做什么（不是替代人，是做重复劳动的过滤）
2. **展示能力**：展示最小闭环已打通（周报→归一化→判断→摘要→台账→清单）
3. **证明可复核**：每项判断都有规则依据，人工可追溯、可推翻
4. **诚实边界**：明确说明当前是"条件性实现"，不是正式版本
5. **收集反馈**：通过演示收集PMO/管理层的正式确认意见

# 真实影子运行阶段判断规则

**版本：** v1.0
**日期：** 2026-04-08
**阶段：** 阶段4Q-T《真实样本影子运行准备》

---

## 判断规则

### 判断维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 阻断性差异 | 最高 | 有则必须暂停 |
| 抽取完整性 | 高 | 黄灯/延期/结项是否完整 |
| 抽取准确性 | 高 | 无明显错误抽取 |
| 格式正确性 | 中 | 输出格式是否符合模板 |

---

## 阻断性差异（必须暂停）

以下任一项出现即暂停：

| # | 阻断项 | 说明 |
|---|--------|------|
| B1 | 关键事项丢失 | 黄灯/延期/结项完全未识别 |
| B2 | 项目名错误 | 项目名称与原始样本完全不符 |
| B3 | 判断方向错误 | 应识别为黄灯但识别为正常 |
| B4 | 完全无法生成输出 | 脚本执行失败 |
| B5 | 输出内容为空 | 归一化结果为空 |

---

## 可接受差异（可观察后继续）

以下差异不阻断，但需记录观察：

| # | 差异项 | 说明 |
|---|--------|------|
| A1 | 格式表述差异 | 措辞与模板略有不同 |
| A2 | 量化指标偏差 | 指标数值与预期有小幅偏差 |
| A3 | 事项数量偏差 | 总数与预期相差≤2项 |
| A4 | 字段缺失 | risk_level等次要字段为空 |
| A5 | 事项归属偏差 | 一个事项被分到两个分类 |

---

## 阶段判断标准

| 结论 | 标准 |
|------|------|
| **通过** | 无阻断性差异 + 抽取完整 + 抽取准确 |
| **小修** | 无阻断性差异 + 存在可接受差异 |
| **暂停** | 有阻断性差异（B1~B5任一）|

---

## 快速判断流程

```
开始判断
↓
存在B1~B5阻断项？
  ├─ 是 → 结论：暂停
  └─ 否 → 下一步
↓
抽取完整且准确？
  ├─ 是 → 结论：通过
  └─ 否 → 结论：小修
```

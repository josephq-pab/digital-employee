## 一、当前样本周报格式（参考基准）

**样本路径：** `samples/samples/weekly_reports/weekly_report_series_latest3.json`

**数据结构：**
```json
{
  "sample_type": "6BLIVE",
  "sample_version": "v2",
  "is_desensitized": true,
  "report_meta": {
    "report_id": "PMO-WR-2025-1222-1226",
    "report_period": "1222-1226",
    "report_date_text": "二零二五年十二月二十六日",
    "reporting_unit": "某总行条线PMO",
    "report_scope": "重点项目推进周报"
  },
  "project_overview": {
    "total_items": 60,
    "status_light": "green",
    "period_key_requests_summary": "...",
    "period_key_alerts_summary": "..."
  },
  "key_requests": [...],
  "key_alerts": [...],
  "new_items": [...],
  "management_summary": "..."
}
```

**必要字段：**
| 字段路径 | 类型 | 说明 | 是否可缺 |
|---------|------|------|---------|
| report_meta.report_period | string | 期数标识，如"1222-1226" | ❌ 不可 |
| report_meta.report_date_text | string | 日期文本 | ✅ 可 |
| project_overview.total_items | int | 事项数量 | ❌ 不可 |
| project_overview.status_light | string | 状态灯：green/yellow/red | ✅ 可 |
| key_requests | array | 重要需求列表 | ✅ 可 |
| key_alerts | array | 重要预警列表 | ✅ 可 |
| new_items | array | 新增事项列表 | ✅ 可 |
| management_summary | string | 管理层摘要 | ✅ 可（系统可生成） |

## 三、收口会上看什么

> 收口会只审议以下5个维度，不讨论其他事项。

### 维度1：输入是否齐套

| 检查项 | 标准 | 本期实际（样本到位后填写） |
|--------|------|------------------------|
| 周报是否收到 | 是，完整 | ⏳ |
| 台账是否收到 | 是，完整 | ⏳ |
| 指标是否收到 | 是，完整或部分 | ⏳ |
| 数据质量是否满足运行要求 | 是，可读可用 | ⏳ |

**结论**：☐ 通过 / ☐ 不通过（阻塞）

---

### 维度2：输出是否完成

| 输出类型 | 是否生成 | 内容是否完整 |
|---------|---------|------------|
| 领导摘要 | ⏳ | ☐ 完整 / ☐ 部分缺失 |
| 事项总台账 | ⏳ | ☐ 完整 / ☐ 部分缺失 |
| 风险清单 | ⏳ | ☐ 完整 / ☐ 部分缺失 |
| 运行日志 | ⏳ | ☐ 完整 / ☐ 部分缺失 |

**结论**：☐ 完成 / ☐ 未完成（阻塞）

---

### 维度3：偏差是否可控

| 检查项 | 标准 | 本期实际（样本到位后填写） |
|--------|------|------------------------|
| 偏差数量 | 不影响整体判断 | ⏳ |
| 偏差严重程度 | 无严重偏差 | ☐ 轻微 / ☐ 中等 / ☐ 严重 |
| 偏差是否已记录 | 是 | ⏳ |

**结论**：☐ 可控 / ☐ 不可控（需处理）

---

### 维度4：是否有价值

| 检查项 | 标准 | 本期实际（样本到位后填写） |
|--------|------|------------------------|
| 是否减轻工作 | 是 | ☐ 明显减轻 / ☐ 略有减轻 / ☐ 无变化 |
| 是否有管理参考价值 | 是 | ☐ 明显有价值 / ☐ 有些价值 / ☐ 无价值 |

**结论**：☐ 有价值 / ☐ 价值不明显（需讨论）

---

### 维度5：是否触发ABC决策

| 检查项 | 结果 |
|--------|------|
| 是否触发ABC决策 | ☐ A（接受）/ ☐ B（修改后接受）/ ☐ C（不接受）/ ☐ 未触发 |
| 若触发，处置是否明确 | ☐ 是 / ☐ 否 |

**结论**：☐ 决策清晰 / ☐ 需进一步决策（讨论处理方式）

## 四、风险点

| 风险点 | 影响 | 缓解措施 |
|--------|------|---------|
| Q01/Q03/Q04迟迟不确认 | 路径无限延长 | 按阻断项催办话术推进 |
| Q01/Q03/Q04部分确认 | 部分P1问题仍blocked | 先处理不依赖该项的P1 |
| P3等可立即修的工作被推迟 | 试跑质量下降 | 明确并行工作优先级 |
| 外部确认要求修改核心规则 | 可能影响原型方向 | 按P0处理，等确认 |

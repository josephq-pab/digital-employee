## 三、收到样本后的填写规范

一旦收到真实样本，按以下规范填写：

| 字段 | 填写规范 |
|------|---------|
| 样本名称 | 业务方提供的文件名 |
| 周期 | 覆盖的周报期间 |
| 来源 | 哪个业务团队提供 |
| 脱敏状态 | 已脱敏 / 部分脱敏 / 未脱敏 |
| 是否含周报 | 是 / 否 |
| 是否含台账 | 是 / 否 |
| 是否含指标 | 是 / 否 |
| 当前处理状态 | 已收到待校验 / 校验中 / 已校验通过 / 已校验未通过 |
| 校验结论 | 通过 / 不合格（附不合格原因）|
| 登记人 | 技术侧登记人姓名 |

## 四、样本更新触发规则

| 触发条件 | 动作 |
|---------|------|
| 收到样本 | 24小时内登记在 S-001 位置，状态改为"已收到待校验" |
| 校验完成 | 更新校验结论和状态 |
| 校验未通过 | 状态改为"已校验未通过"，通知业务方重新提供 |
| 校验通过 | 状态改为"已校验通过"，可进入影子运行准备 |

**反馈类型选项**（只需填选项编号）：
- **A类**：页面体验问题（操作不响应、布局乱、字体看不清等）
- **B类**：表达问题（文字看不懂、判断依据不清晰、语义错误等）
- **C类**：流程问题（不知道下一步该做什么、入口不清晰等）
- **D类**：非阻断建议（有改进想法但不紧急）
- **E类**：阻断问题（完全无法使用，必须先修）

**优先级**：高 / 中 / 低

## 一、校验状态定义

| 状态 | 说明 |
|------|------|
| 待校验 | 样本已收到，待执行校验 |
| 校验中 | 正在执行校验 |
| 校验通过 | 全部校验项通过，可进入影子运行 |
| 校验未通过 | 至少一项校验项不通过 |

---

## 二、校验项（全部通过才可进入影子运行）

| 校验类型 | 校验项 | 标准 | 结果 |
|---------|-------|------|------|
| period清晰 | start_date / end_date / report_period 存在 | YYYY-MM-DD 格式 | 通过/不通过 |
| 周报结构 | request_lines / alert_lines / raw_text / overview 字段齐全 | 字段类型正确 | 通过/不通过 |
| 内容有效性 | request_lines + alert_lines 总数 > 0 | 至少1条事项 | 通过/不通过 |
| raw_text有效性 | raw_text 长度 > 100字符 | 有足够正文 | 通过/不通过 |
| 脱敏检查 | 项目名不包含真实名称 | 已脱敏 | 通过/不通过 |
| JSON格式 | 可被 json.loads() 解析 | 无语法错误 | 通过/不通过 |

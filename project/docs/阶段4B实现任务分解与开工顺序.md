## 三、开工顺序（单Agent串行版）

```
T01（样本装载）
  └─ T02（输入读取）
      └─ T03（事实抽取）
          └─ T04（规则判断）
              ├─ T05（领导摘要生成）
              │   └─ T09（冒烟测试执行）
              │       └─ T10（演示样例固化）
              ├─ T06（事项总台账）← 可与T05并行
              └─ T07（风险清单）   ← 可与T05并行
                  └─ T08（人工确认回填）
```

---

## 四、关键路径分析

```
关键路径：T01 → T02 → T03 → T04 → T05 → T09 → T10
总任务包数：10个
关键路径长度：7个串行任务包
```

**任何关键路径上的任务延迟，都会直接延长整体工期。**

## 二、三级真相源体系

| 层级 | 文档 | 职责 |
|------|------|------|
| **第一入口（边界真相源）** | 一期原型总览.md | 定义边界、原则、不做什么。新Agent第一份必读。 |
| **第二入口（字段/规则真相源）** | 输入输出数据契约.md + 一期术语与判断口径.md | 定义字段口径和判断规则。所有实现以此为准。 |
| **第三入口（阶段推进真相源）** | 阶段推进总表.md | 统一记录每阶段门控条件。不得散落在报告里。 |

## 一、待命状态说明

**当前状态：影子运行处于待命状态，不是"已启动"。**

| 状态 | 说明 |
|------|------|
| 待命 | 触发条件未满足，未启动影子运行 |
| 可启动 | 全部条件满足，等待触发指令 |
| 运行中 | 影子运行执行中 |
| 已完成 | 影子运行已完成 |

|------|------|

## 三、启动前仍需满足的条件

**触发条件（必须全部满足）：**
1. 真实样本到位（≥ 3期）
2. 样本校验通过（全部6项）
3. 人工确认对接人已指定
4. 影子运行前最终检查全部通过

---

## 四、样本合格后的最小动作集

一旦样本合格（即样本到位 + 校验通过），执行以下最小动作集：

| 步骤 | 动作 | 执行人 | 时效 |
|------|------|--------|------|
| 1 | 执行《影子运行前最终检查表》 | 技术侧 | 即时 |
| 2 | 全部通过后，通知业务方确认对接人 | 技术侧 | 即时 |
| 3 | 确认对接人已指定 | 业务侧 | 即时 |
| 4 | 打开《真实试运行结果登记表》| 技术侧 | 即时 |
| 5 | 正式触发影子运行 | 技术侧 | F1~F4全部通过后 |

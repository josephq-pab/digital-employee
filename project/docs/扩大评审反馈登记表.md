# 扩大评审反馈登记表

**版本：** v1.0
**日期：** 2026-04-08
**阶段：** 阶段4M《扩大评审执行与真实样本接入启动》

---

## 一、使用说明

本登记表用于记录评审对象对扩大评审的意见。每次评审会议后，填写一条记录。结论标记为"待反馈"时表示评审已完成但意见尚未正式登记。

---

## 二、字段说明

| 字段 | 说明 |
|------|------|
| 反馈序号 | 自动编号（FB-001开始）|
| 反馈日期 | 评审会议日期 |
| 反馈人 | 评审参与人 |
| 反馈类型 | 认可/有条件认可/暂缓/拒绝 |
| 反馈内容 | 评审意见摘要 |
| 影响扩大结论 | 是/否 |
| 后续动作 | 针对反馈意见的具体处理动作 |
| 登记人 | 记录人 |

---

## 五、结论判定规则

| 反馈类型 | 综合结论 | 说明 |
|---------|---------|------|
| 全部认可 | 通过 | 可进入分支A（评审通过）|
| 有条件认可（条件可满足）| 附条件通过 | 可进入分支A，但需满足记录的条件 |
| 有条件认可（条件不可满足）| 暂缓 | 进入分支C（暂缓）|
| 存在暂缓意见 | 暂缓 | 进入分支C |
| 存在拒绝意见 | 拒绝 | 暂停，等待进一步指示 |

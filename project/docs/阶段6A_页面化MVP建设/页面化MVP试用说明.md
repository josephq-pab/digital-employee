## 一、系统定位

本系统是**数字员工并行运行内部工具**，不是替代人工的自动化系统。

**定位**：与现有PMO周报机制**并行运行**，为同一份数据提供第二个视图，辅助而不替代人工决策。

**输出**：领导摘要 / 事项总台账 / 风险清单 / 复核记录

**人工职责**：最终管理决策 / 与业务方沟通 / 风险升级处置

## 三、快速试用（5分钟上手）

### 第1步：切换到真实样本视图

页面默认显示"真实样本：0414-0418"（当前周期）。

点击顶部按钮可切换：
- **真实样本：0414-0418** ← 当前默认
- **样本1：1117-1121**（延期/黄灯/新增）
- **样本2：1208-1212**（结项/黄灯并存）

### 第2步：查看领导摘要

点击「📋 领导摘要」，查看本周期整体判断。

### 第3步：查看事项总台账

点击「📊 事项总台账」，按类型筛选（关注/黄灯/延期/新增/结项）。

### 第4步：查看风险清单

点击「⚠️ 风险清单」，查看当前关注/黄灯类事项。

### 第5步：体验复核流程

点击「✅ 复核与ABC决策」，了解本周期复核状态。

## 八、常见问题

**Q：为什么台账只有5项？**
A：本期台账为数据提供方选取的重点事项，非全量58项，全量详情请参考原始周报正文。

**Q：关注和中灯有什么区别？**
A："关注（中）"表示需要关注但未到红灯程度的事项，本系统中以橙色标签区分于"黄灯"。

**Q：指标偏离8.2亿是什么意思？**
A：M-001汽车零部件贷款余额净增的目标是15亿元，当前实际净增6.8亿元，缺口8.2亿元。

**Q：系统会自动帮我做决策吗？**
A：不会。系统仅提供辅助视图，最终管理决策由人工做出。

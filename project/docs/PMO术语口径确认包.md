# PMO术语口径确认包

**版本：** v1.0
**发出方：** 数字员工项目组

**用途：** 请PMO确认8个核心术语的定义和5条判断规则是否与业务理解一致

---

## 一、确认包概述

本包用于请求PMO确认《一期术语与判断口径》中的核心术语和判断规则。
确认后，术语口径将被正式冻结，阶段4开发以此为准。

## 五、仍允许在试点期微调的内容

以下内容在试点运行中允许小幅微调，不需要走正式变更流程：

| 可微调项 | 微调范围 | 前提条件 |
|---------|---------|---------|
| 关键词列表 | 新增同义词，不改变规则逻辑 | PMO书面确认 |
| 阈值（如"≥3天"） | 根据试点数据调整 | PMO书面确认 |
| 建议动作措辞 | 文字优化，不改变动作类型 | Agent记录即可 |

---

## 六、未确认会阻断什么

若PMO未确认术语口径：

| 阻断项 | 影响 |
|--------|------|
| 阶段4无法进入 | 开发基准不明确 |
| 冒烟测试无法最终验收 | 测试用例的预期结果无法与业务对齐 |
| 演示版无法正式制作 | 判断逻辑未获业务背书 |

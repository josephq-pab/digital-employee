## 五、样本格式差异说明

**November/December 样本包结构**（alert_lines/request_lines 字符串格式）：
- `alert_lines: [string, ...]` - 每条是完整字符串，需解析抽取出项目名
- `request_lines: [string, ...]` - 同上
- 无 `key_alerts`/`management_summary` 对象数组

**原型引擎期望格式**（key_alerts/management_summary 对象格式）：
- `key_alerts: [{project_name, alert_level, raw}, ...]`
- `management_summary: {items_requiring_leadership_attention, delayed_items, ...}`

**解决方案**：创建 `src/loaders/november_adapter.py` 作为格式适配层，在不修改 src/ 核心代码的前提下实现数据转换。

## 二、G13详情（ABC选项tooltip）

**改前：**
- ABC三个选项有简短说明
- 无hover详细提示

**改后：**
- A（接受）：hover显示"选此项的条件：系统输出的结论卡/摘要/台账/风险清单完全可接受，无需任何修改即可使用"
- B（修改后接受）：hover显示"选此项的条件：结论基本可用，但存在部分数据或分类需要人工核实或修改后使用"
- C（不接受）：hover显示"选此项的条件：系统输出存在重大错误或遗漏，不适合作为管理参考材料"

**技术实现：**
- HTML：data-tooltip属性 + title属性
- CSS：[data-tooltip]:hover::after 显示浮层
- 位置：`index.html` 第849/854/859行

**验证：** 冒烟测试 7/7 PASS ✅

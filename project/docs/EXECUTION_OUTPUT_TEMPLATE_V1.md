# 执行输出模板 V1

**版本**：V1（2026-04-09）  
**用途**：执行层面（PMO/业务方）台账和清单的标准输出格式  

---

## 格式结构

### 完整台账格式

```
【标题】
事项总台账 + 周期 + 生成日期

【筛选栏】
类型标签（结项/推进中/黄灯/逾期）

【汇总行】
- 总事项数
- 各状态数量

【表格】
| # | 事项名称 | 类型 | 主责部门 | 计划节点 | 系统判断 | 风险 | 管理建议 |
```

### 风险清单格式

```
【标题】
风险清单 + 周期 + 生成日期

【风险分级标准说明】
- 黄灯 = 节点超期或进度严重滞后
- 逾期 = 已过节点未完成
- 指标偏离 = 量化指标持续偏离目标

【风险明细表】
| 风险ID | 事项 | 风险类型 | 风险说明 | 判断依据 | 处置建议 |
```

### 风险清单

**风险分级标准**：
- 🔴 黄灯 = 节点超期或进度严重滞后，需确认节点和资源
- 🔴 逾期 = 已过节点未完成，需立即处置
- 🔵 指标偏离 = 量化指标持续偏离目标，需分析原因

| 风险ID | 事项 | 风险类型 | 风险说明 | 判断依据 | 处置建议 |
|--------|------|---------|---------|---------|---------|
| R-001 | 学校行业 | 黄灯 | 储备不足，逾期近3个月 | 节点9月末→已过 | 确认储备数量 |
| R-002 | 汽车行业 | 黄灯 | 中试未启动 | 节点11月→未启动 | 推进12月启动 |
| R-003 | 双链融合 | 逾期 | 原定11月上旬汇报 | 节点已过1个月+ | 已排期12/18 |
| R-004 | 并购贷款 | 指标偏离 | -40亿 vs 目标-180亿 | 偏离超7倍 | 分析原因 |

## 一、触发总则

| 原则 | 说明 |
|------|------|
| 不得提前触发 | 不得在Q01/Q03/Q04未正式确认的情况下触发正式试跑 |
| 部分确认可部分触发 | Q01单独确认 → 解锁对应问题处理，但不触发"正式启动" |
| 全部确认触发正式试跑 | Q01+Q03+Q04全部确认 → 门控全🟢 → 发布启动通知 → 正式启动 |

```

    ↓

    ↓

    ↓

    ↓
第六步：进入首周SOP节奏
```
